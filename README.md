

# 📚 Repositório de Exercícios Java

Este repositório contém uma série de exercícios práticos em **Java**, organizados por aulas e temas. Cada exercício está implementado em um arquivo `.java` e visa ajudar no aprendizado e prática da linguagem.

---

## 🗂️ Estrutura do Projeto

### 1️⃣ **Pasta `helloworld`**  
- **Arquivo `HelloWorld.java`**: Exibe a famosa mensagem **"Hello World"** na tela.

### 2️⃣ **Pasta `aula_02`**  
- **Arquivo `Calculadora.java`**: Implementa uma calculadora básica para soma, subtração, multiplicação, divisão e cálculo de módulo.
- **Arquivo `Exer01.java`**: Calcula o novo salário após o recebimento de um abono.
- **Arquivo `Exer02.java`**: Calcula a média de quatro notas inseridas pelo usuário.
- **Arquivo `Exer03.java`**: Calcula o salário líquido com base no salário bruto, adicional noturno, horas extras e descontos.
- **Arquivo `Exer04.java`**: Calcula a diferença entre dois produtos com base nos números inseridos.
- **Arquivo `Operadores.java`**: Demonstra o uso de operadores aritméticos, de concatenação e formatação.

### 3️⃣ **Pasta `aula_03`**  
- **Arquivo `Exer01.java`**: Compara a soma de dois números com um terceiro e exibe o resultado.
- **Arquivo `Exer02.java`**: Verifica se um número é positivo ou negativo e se é par ou ímpar.
- **Arquivo `Exer04.java`**: Classifica um animal baseado em sua alimentação (herbívoro, carnívoro, etc.).
- **Arquivo `Exer05.java`**: Calcula o total de uma compra com base no produto e quantidade escolhidos.
- **Arquivo `Exer06.java`**: Calcula o salário reajustado de um funcionário com base no cargo.
- **Arquivo `Exer07.java`**: Calculadora que realiza operações de soma, subtração, multiplicação e divisão.
- **Arquivo `Exer08.java`**: (Descrever exercício, se necessário.)

### 4️⃣ **Exercícios de Aulas Diversas**  
- **Exer08**: Simulação de um sistema bancário 💳  
  - Simula operações simples de conta bancária, como consulta de saldo, saque e depósito.
  
- **PlanoSaude**: Cálculo de plano de saúde baseado na idade 👵👶  
  - Calcula o custo mensal de um plano de saúde com base na faixa etária.

- **PlanoSaudeV2**: Cálculo de plano de saúde usando faixas etárias 🧑‍⚕️  
  - Apresenta um cálculo mais detalhado com faixas etárias divididas em intervalos específicos.

## 📚 Exercícios de Java - Aula 02 💻

### 1️⃣ **Exer01** - Calculadora de Salário 💵  
Calcula o novo salário após o recebimento de um abono. O exercício deve mostrar o valor do abono, o salário anterior e o novo salário.

### 2️⃣ **Exer02** - Média de Notas 📊  
Calcula a média de quatro notas inseridas pelo usuário e informa se o aluno foi aprovado ou reprovado com base na média calculada.

### 3️⃣ **Exer03** - Salário Líquido 🧾  
Calcula o salário líquido de um funcionário levando em consideração o salário bruto, o valor do adicional noturno, horas extras e descontos.

### 4️⃣ **Exer04** - Diferença entre Produtos 💲  
Solicita ao usuário dois valores representando o preço de dois produtos e exibe a diferença entre esses valores.

### 5️⃣ **Operadores** - Operações Aritméticas ➗  
Exemplo de uso de operadores aritméticos e de concatenação. O exercício permite ao usuário calcular a soma, subtração, multiplicação, e divisão de dois números inteiros.

## 📚 Exercícios de Java - Aula 03 💻

### 1️⃣ **Exer01** - Comparação de Números 🔢  
Compara a soma de dois números com um terceiro e exibe uma mensagem indicando se o terceiro número é maior, menor ou igual à soma dos outros dois.

### 2️⃣ **Exer02** - Par ou Ímpar 🔄  
Verifica se o número inserido pelo usuário é positivo ou negativo e se é par ou ímpar, exibindo a resposta.

### 3️⃣ **Exer04** - Classificação de Alimentação 🍖🥦  
Classifica um animal com base na sua alimentação: herbívoro, carnívoro ou onívoro, dependendo da escolha do usuário.

### 4️⃣ **Exer05** - Total da Compra 🛒  
Calcula o total de uma compra de supermercado com base no valor unitário dos itens e na quantidade comprada, considerando a inclusão de impostos.

### 5️⃣ **Exer06** - Reajuste de Salário 💼  
Calcula o novo salário de um funcionário com base no cargo. Se o cargo for "Gerente", aplica um reajuste de 10%. Para outros cargos, aplica um reajuste de 5%.

## 📚 Exercícios de Java - Aula 04 💻

### 1️⃣ **Exer01** - Múltiplos de 3 e 5 🧮  
Solicita ao usuário dois números inteiros representando um intervalo e exibe os múltiplos de 3 e 5 dentro desse intervalo.

### 2️⃣ **Exer02** - Contagem de Números Pares e Ímpares 🔢  
Pede ao usuário para inserir 10 números e exibe a quantidade de números **pares** e **ímpares** inseridos.

### 3️⃣ **Exer03** - Faixa Etária de Pessoas 👶👵  
Coleta idades de diversas pessoas até um número negativo ser inserido. Exibe quantas pessoas têm menos de 21 anos e quantas têm mais de 50 anos.

### 4️⃣ **Exer04** - Pesquisa de Perfil Profissional 🧑‍💻  
Realiza uma pesquisa sobre perfil profissional, coletando dados como idade, identidade de gênero e função no trabalho. Faz análises com base em faixas etárias e funções, além de calcular a média de idade.

### 5️⃣ **Exer05** - Soma de Números Positivos ➕  
Solicita ao usuário que insira números inteiros e soma apenas os valores **positivos**. A execução termina quando o número zero é inserido.

### 6️⃣ **Exer06** - Média de Múltiplos de 3 📊  
Soma os múltiplos de 3 inseridos pelo usuário e exibe a média desses múltiplos.

### 7️⃣ **Pesquisa** - Preferências Esportivas ⚽🏀  
Coleta informações sobre a idade e o esporte favorito dos participantes, com análise de preferências específicas de faixas etárias.

### 8️⃣ **TabuadaV2** - Tabela de Multiplicação 🧑‍🏫  
Solicita um número ao usuário e exibe a **tabuada** desse número de 1 até 10 de forma interativa.

## 📚 Exercícios de Java - Aula 05 💻

### 1️⃣ **Exer01** - Busca Linear 🔍  
Realiza a busca de um número em um vetor. O sistema retorna a posição do número no vetor ou informa se não foi encontrado.

### 2️⃣ **Exer03** - Diagonais de Matrizes 🔢  
Imprime os elementos das diagonais principal e secundária de uma matriz 3x3, além de calcular a soma de cada uma.

### 3️⃣ **Exer04** - Cálculo da Média de Notas 📊  
Permite o cadastro das notas de dois alunos e calcula a média geral de cada um.

### 4️⃣ **Matrizes** - Matrizes e Manipulação de Dados 🧮  
Trabalha com matrizes, incluindo matrizes irregulares, e permite a manipulação interativa de dados.

### 5️⃣ **Vetores** - Manipulação de Vetores e Busca 🧑‍💻  
Manipula vetores (arrays), incluindo ordenação, busca binária e localização de elementos.

## 📝 Como Usar

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/repositorio-java.git
   ```

2. **Compile e execute os arquivos:**
   Para executar um exercício, compile o arquivo `.java` correspondente e execute-o.

   Exemplo:
   ```bash
   javac HelloWorld.java
   java HelloWorld
   ```
